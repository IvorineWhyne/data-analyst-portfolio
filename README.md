## 📈 How to Explore

1. **Excel Files**: Open directly to view raw data and analysis
2. **Power BI Files (.pbix)**: 
   - Download [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free)
   - Open the `.pbix` files to interact with live dashboards
3. **Exported Reports**: View static PDFs and images in `power-bi/report-exports/`

## 📝 Getting Started

- Start with `power-bi/README.md` for detailed project descriptions
- Review individual project folders for methodology and insights
- Check exported reports for quick visual summaries
